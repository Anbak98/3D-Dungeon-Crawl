# 3D-Thired-Aspect-Ball
The simple 3d example

This project include some algorithms.
- Third-aspect camera
- Ball movement
- Jump pad & Charging-Jump pad
- Inventory system
- Speed up item
- Moving Platfom
- Dot damage player object
- Laser Trap
- Raycast detector
- Ground & Wall check
